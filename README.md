[MintHCM](https://minthcm.org/) is a professional tool for Human Capital Management. It's based on two popular, well-known business applications: SugarCRM Community Edition and SuiteCRM. This is why we often say that **MintHCM is CRM reinvented**. You all know how much goes into technological development of this type of business software‚Ä¶ <br> **That's why we need your help.**<br> Thanks to the open code of both SugarCRM CE and SuiteCRM we were able to reintroduce some features, redesign other, and provide brand new system to the users. The code of MintHCM remains open so feel free to use it.
Have a look at the official [MintHCM website](https://minthcm.com/) for a list of MintHCM's features and use cases.
